- **Learning:** 
  Fixed point numbers are represented in binary very simply. This is a standard expression and follows by converting to base 2 from base 10. Floating point values can be represented in binary but it is usually through and implicit expression. We define how many integer bits and fractional bits and then the implecit decimal point can be determined. By doing so we can perform arithmatic with the values.

  Sequential circuit components liek counters and shift registers are important.
  Counters are created by simply incrementing a local variable when the clock edge is triggered. This can be accompanied by a reset and enable to dictate and influence the state of the counter.
  Shift registers are a string of registers which can help to delay the movement of data they are attached and move the values to the next register every clock cycle.
