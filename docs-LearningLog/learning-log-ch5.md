- **Learning:** 
  Memory has developed alot since it was first introduced. RAM and ROM were initially distriguished by their differing ability to write and read data, but now adays you can do both with either. ROM is nonvolatile. It does not disappear when we remove power, RAM is volatile so it will lose value when power is removed. We also have DRAM, SRAM, and others like PROM and EPROM. They all have differening capabilities. The major difference is the time it takes to extract and the number of transistors needed. Flip flops are the fasted but are most transistor expensive, DRAM is the least expensive but take the most amount of time, SRAM is inbetween.

  Other systems are created like PLA which are configurable and can be used to generate any time of combiantional logic. FPGA are a better version and are an array of configurable logic which can be used to develop combinational and sequential logic. This is an industry standard to developing chips which are used for specific applications. The prototyping tool is an FPGA.

  Shift registers are a string of registers which can help to delay the movement of data they are attached and move the values to the next register every clock cycle.
